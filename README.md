# Diced

A modern five-dice score game prototype for the web.

## What works now

- 2-4 player pass-and-play games
- animated rolling with held dice
- 13 score categories per player
- live scoreboard and game-over winner state
- room-code themed lobby that is ready to connect to real sockets

## Next online-game steps

1. Add an Express + Socket.IO game server.
2. Move dice rolls and scoring validation to the server.
3. Let players create and join rooms by code.
4. Sync turns, held dice, scorecards, and game-over results to all clients.

Live site: https://5diced.netlify.app/
